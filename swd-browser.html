Can’t remember what that one event does? Looking for cards with the “Ambush” keyword? Need to see exactly which cards are available in the Trilogy format?

Let the SWD Browser app help!

  • Browse every single card available in the Star Wars Destiny game (over 1,400 cards as of SoH!)
  • View each card in a detail view, including the card image if available
  • Check what starter set a card is available in
  • Search for cards by name
  • Filter by affiliation, faction, set, type, subtype, keyword, and rarity

All this for FREE!
